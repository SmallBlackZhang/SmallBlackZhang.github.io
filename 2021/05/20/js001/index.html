类型分类

最新的 ECMAScript 标准定义了 8 种数据类型:
分为原始数据类型（值类型）和引用数据类型（对象类型）
最基本原始类型包括以下 7 个：

  • Number
  • Boolean
  • String
  • Null(特殊)
  • Undefined
  • Symbol
  • Bigint(es2020新增)

引用类型统称为 Object 类型,常用的Object 类型有以下 9 个：

  • Function
  • Array
  • Date
  • Math
  • RegExp
  • Map
  • Set
  • JSON
  • ArrayBuffer

值类型取值范围

类型 取值范围
Number -(2^53 -1) 到 2^53 -1, +Infinity，-Infinity 和 NaN
Boolean true false
String 任意字符串
Null null
Undefined undefined
Symbol 符号类型值是唯一的并且是不可修改的
Bigint 仅在合理预期大于 2^53 的值时才使用 BigInt 值

判断数据类型

  • 严格相等===
  • typeof
  • instanceof
  • Object.prototype.toString.call
  • constructor
